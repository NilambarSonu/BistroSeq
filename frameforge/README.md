# BistroSeq 🍷

**BistroSeq** is an elegant, high-performance browser-based tool for converting cinematic videos into professional-grade PNG or JPEG image sequences. 

### Why BistroSeq?

*   **120 FPS Support:** Extract frames at up to 120 frames per second.
*   **Artisanal Quality:** Beautifully crafted UI inspired by high-end bistro menus.
*   **100% Private:** Your video never leaves your browser. All processing is "House Local".
*   **Blazing Fast:** Uses WebCodecs hardware acceleration when available.
*   **Lossless Exports:** Support for high-fidelity PNG sequences.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technology Stack

*   **Next.js 16 (App Router)**
*   **Tailwind CSS v4**
*   **Framer Motion**
*   **JSZip**
*   **Lucide React**
*   **Sonner**

## Process & Philosophy

BistroSeq is built with the "BistroMenu Design System", focusing on elegant typography, luxurious spacing, and an artisanal feel. All processing is done locally via the Canvas API and WebCodecs, ensuring your "ingredients" (videos) stay on your device.
